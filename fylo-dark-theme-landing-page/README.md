# Description
This is a <i>fylo-dark-theme-landing-page</i> challenge from <a href="https://www.frontendmentor.io/challenges/fylo-dark-theme-landing-page-5ca5f2d21e82137ec91a50fd" target="_blank">Frontend Mentor</a>.
I have used <b>HTML5</b>, <b>CSS Flexbox</b>, <b>Grid</b>, and <b>Media Queries</b> to make it fully responsive across all screen sizes. Have also used a bit of <b>JavaScript</b> for email validation.
